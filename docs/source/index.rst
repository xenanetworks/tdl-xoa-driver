This documentation is intended for test specialists who are developing and running automated test scripts/programs using Xena test equipment. The XOA Python API is a Python-based driver that provides a high-level interface to the Xena test equipment, allowing users to easily control and configure the hardware for testing purposes.

This documentation covers how to use the XOA Python API, including how to install it, examples of how to use it, and detailed API documentation. It's organized by API function, with descriptions, examples, and expected results.
