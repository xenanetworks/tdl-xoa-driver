``hlfuncs`` package
===============================

``hlfuncs`` package provides high-level abstraction functions on top of the object-oriented APIs in HL-API, aiming to help you simplify code logics and increase readability and maintainability. ``hlfuncs`` package consists of sub-modules where functions are grouped based on functionalities.
