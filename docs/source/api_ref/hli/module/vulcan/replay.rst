PCAP Replay
=========================

.. note::

    For Vulcan module only.


List Replay Files
--------------------


.. code-block:: python

    await module.replay.file.list.get()


List Replay BSON Files
----------------------


.. code-block:: python

    await module.replay.file.list_bson.get()


Delete Replay File
--------------------


.. code-block:: python

    await module.replay.file.delete.set()
